# Dex

Dex : The data explorer is a data visualization tool written in java/javafx capable of powerful ETL and publishing D3
and other HTML visualizations.

# Requirements

 - Java 8

# Installation

git clone https://github.com/PatMartin/Dex.git

# Running

cd Dex
java -jar Dex.jar
